# FJL Painting Website

Static website for `fjlpainting.com`.

## Files

- `index.html` - main page
- `assets/styles.css` - all styles
- `assets/main.js` - mobile menu behavior

## Before launch

Replace these placeholders:

- `Sydney and surrounding suburbs`, if the service area should be more specific
- Real portfolio photos in the portfolio section
- Optional: replace mailto quote handling with Formspree, Netlify Forms, or a Vercel API

## Deploy with GitHub Pages

Recommended fast path:

1. Push this folder to a GitHub repository.
2. In GitHub, open the repository settings.
3. Go to Pages.
4. Set the source to GitHub Actions.
5. The included workflow will publish the static site.
6. After the GitHub Pages URL works, point `fjlpainting.com` to GitHub Pages in DNS.
